# Onlyway Venture Operating System

Questa cartella descrive il control plane locale con cui Onlyway trasforma un input fondatore in una Opportunity tracciabile, una Thesis versionata, esperimenti interni e un dossier decisionale. Non è un sistema di investimento autonomo e non autorizza azioni esterne.

Flusso autorevole:

`Founder policy → evidence map → opportunity → deterministic scorecard → thesis → economics → experiment portfolio → launch pack → Fabio decision`

I confini sono obbligatori: Opportunity, Thesis, Experiment, Venture, Launch ed esecuzione esterna sono entità diverse. Nessuna transizione attiva una Venture o spende capitale senza una decisione di Fabio legata a versione e fingerprint.

Stati di sicurezza permanenti in questa release:

- `EXTERNAL_ACTION_LOCKED`
- `PROPOSAL_ONLY`
- `PUBLICATION_LOCKED`
- input economico assente: `NOT_AVAILABLE`
- scelta materiale assente: `FOUNDER_INPUT_REQUIRED`
