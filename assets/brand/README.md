## Metodo Veloce

`metodo-veloce-logo-original.jpg` is Fabio's immutable source asset. Its SHA-256,
dimensions, provenance and non-modification rule are recorded beside it in
`metodo-veloce-logo-original.json`.

The only allowed derivative is
`derived/metodo-veloce-logo-overlay-technical.png`: a lossless local crop used
exclusively as a final local overlay. Its manifest links it to the original hash.
